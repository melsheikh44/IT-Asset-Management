# IT-Asset-Management

A simple Python app for managing IT assets using a local SQLite database with basic Functions below:

- Viewing all Assets.
- Find an Assets.
- Add a new Asset.
- Update an Asset.
- Delete an Asset.
- Importing Bulk of assets data from a CSV file via (Bulk_Add_Assets.csv).
